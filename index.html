About

Welcome to the course on AI/ML Lab! This comprehensive and engaging course explores the various methods of designing and implementation of AI and Machine Learning models in the areas of image processing,speech processing,NLP and game playing.You will also learn about how to develop applications with AI/ML development board/kits.

Syllabus

  • Design and implementation of Al and machine learning models for image processing
  • Design and implementation of Al and machine learning models for speech processing
  • Design and implementation of Al and machine learning models for NLP
  • Design and implementation of Al and machine learning models for Game playing
  • Deploy machine learning models on mobile
  • Developing applications with Al/ML development board/kits
